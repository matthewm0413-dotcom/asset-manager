# NQ1! FVG Alert System Setup Instructions

## Discord Webhook Configuration
1. Go to your Discord server.
2. Navigate to **Server Settings** -> **Integrations**.
3. Click on **Webhooks** and then **Create Webhook**.
4. Name your webhook, select the channel to send messages to, and click **Copy Webhook URL**.
5. Paste this URL in your configuration settings.

## TradingView Pine Script Setup
1. Open TradingView and load the desired chart.
2. Click on **Pine Editor** at the bottom.
3. Paste your Pine Script code into the editor.
4. Save the script and add it to your chart.
5. Set alerts based on the conditions defined in your Pine Script.

## Deployment Guidelines
1. Make sure you have the necessary environment set up (e.g., Node.js, Python).
2. Clone the repository:
   ```
   git clone https://github.com/matthewm0413-dotcom/asset-manager.git
   ```
3. Navigate to the project directory:
   ```
   cd asset-manager
   ```
4. Install necessary dependencies:
   ```
   npm install
   ```
5. Configure your settings as instructed in the configuration files.
6. Start the application:
   ```
   npm start
   ```
7. Monitor logs to ensure everything is working correctly.

Feel free to reach out for any questions regarding the setup process.
